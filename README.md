# 3X-UI 汉化版

一个纯净的 3X-UI 中文安装脚本，基于 mhsanaei/3x-ui 原版，无广告，无付费功能。

## 快速安装

```bash
bash <(curl -Ls https://raw.githubusercontent.com/a820820/x-ui/main/install.sh)
```

## 功能特点

- ✅ 完全汉化的安装界面
- ✅ 无广告，无付费功能
- ✅ 基于原版 mhsanaei/3x-ui
- ✅ Web 面板内置中文支持
- ✅ 完整的代理管理功能

## 使用说明

### 安装后设置中文

1. 登录 3X-UI Web 面板
2. 在设置中找到语言选项
3. 选择 "简体中文"
4. 刷新页面

### 常用命令

```bash
x-ui              # 进入管理菜单
x-ui start        # 启动面板
x-ui stop         # 停止面板
x-ui restart      # 重启面板
x-ui status       # 查看状态
x-ui update       # 更新面板
```
